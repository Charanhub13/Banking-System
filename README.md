# 🏦 Banking System (Core Java)

## 📌 Overview
This is a console-based Banking System built using Core Java.  
It demonstrates Object-Oriented Programming and Collections with real-world banking features.

## 🚀 Features
- Create Account (Savings / Current)
- Deposit & Withdraw Money
- Check Balance
- View All Accounts
- Transaction History with Timestamp
- Input Validation

## 🧠 Concepts Used
- OOP (Encapsulation, Inheritance, Polymorphism, Abstraction)
- Collections (HashMap, ArrayList)
- Java Time API (LocalDateTime)

## 🛠️ Tech Stack
- Java
- Eclipse IDE

## ▶️ How to Run
```bash
javac src/com/BANK/*.java
java com.BANK.Main
